# Teacher Guide

## Goal

Help students build practical intuition for robotics control and reinforcement learning using simulation before hardware access.

## Suggested Class Flow

1. 10-15 min mini-lecture
2. 25-35 min guided coding and simulation
3. 10-15 min discussion and reflection

## Teaching Tips

- Start with visible outcomes (`run_reacher.py`) to keep engagement high.
- Keep one concept per class: action space, reward, policy, evaluation.
- Encourage students to modify one parameter at a time.
- Save all project outputs in `runs/logs`.

## Assessment Ideas

- Weekly simulation reflection journal
- Mid-course checkpoint demo (Week 5)
- Final team demo with reproducible script and short report

## Future Hardware Transition

When students are ready, map simulation concepts to:

- Ark framework modules for orchestration and robotics workflows
- Unitree G1 SDK interfaces for sensing and control pipelines

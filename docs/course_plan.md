# 8-Week Robotics Course Plan

## Week 1 - Introduction to MuJoCo Simulation

- Install Python environment and dependencies
- Run the first simulation script
- Explain observation, action, and reward

## Week 2 - Robot Arm Control (Reacher)

- Explore `Reacher-v5`
- Test random vs simple hand-built control ideas
- Analyze episode rewards

## Week 3 - Robot Balance (Hopper)

- Explore `Hopper-v5`
- Study instability and recovery
- Plot reward trends across runs

## Week 4 - Multi-leg Robots (Ant)

- Introduce `Ant-v5` as an optional extension
- Compare locomotion strategies across robots
- Discuss state/action complexity growth

## Week 5 - Humanoid Simulation

- Explore `Humanoid-v5`
- Discuss high-dimensional control challenges
- Prepare RL experiment setup

## Week 6 - Group Projects

- Teams define a control objective
- Run structured experiments
- Keep logs and observations

## Week 7 - Integration

- Improve policy quality and consistency
- Connect simulation lessons to real humanoid robotics concepts
- Review Ark framework and Unitree G1 SDK pathways

## Week 8 - Final Demo

- Team presentations and live simulation demos
- Evaluate technical approach and learning outcomes
- Class retrospective and next steps
